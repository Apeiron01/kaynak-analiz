Usage Logic

Put the files under `assets/images/`

Save the images as, for example, `assets/images/ls-01-hero.webp`. Then replace the placeholder content with an `img` tag.

LS-01

Hero Campaign Image

Main cyber campaign image. An abstract security form, high contrast portrait or technological composition works well.

  • Area: Home / Case 01
  • Recommended ratio: 4:5 or 5:6
  • Filename: `assets/images/ls-01-hero.svg`
LS-02

Link and Message Security Image

A phone screen, message card, QR, interface that feels like a fake link, or a mobile security composition would be appropriate.

  • Area: Home / Case 02
  • Recommended ratio: 16:10 or 4:3
  • File name: `assets/images/ls-02-link-mesaj.svg`
LS-03

Corporate Security Visual

A more business-oriented visual, such as a dashboard, brand safety screen, corporate team, or cyber operations panel.

  • Area: Home / Case 03
  • Recommended ratio: 16:10 or 3:2
  • File name: `assets/images/ls-03-marka-guvenligi.svg`

What to pay attention to?

  • Use `webp` if possible.
  • For large files, try not to exceed the 250-400 KB band.
  • Do not use very small text within the image; disappears on mobile.
  • Leave a breath on the edges of areas where text will pass.
