privacy policy

At Lumina Siber, we offer tools and analysis services that help users assess digital security risks. Privacy and data security are among our main priorities.

This data may be processed to provide platform services, perform security analyses, prevent abuse, respond to user requests and improve service quality.

Personal data is stored only for as long as necessary. Communication requests may be retained for a reasonable period of time; Analysis data is not stored for the purpose of creating a permanent database.
